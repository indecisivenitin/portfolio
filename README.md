# Nitin Yadav Portfolio

Welcome to my personal portfolio website! This project showcases my skills, projects, and experiences as a web developer. It’s built using **React** and **Vite**, and deployed on **Netlify** for easy access.

**Live Demo:** [https://portfolio-nitin31.netlify.app/](https://portfolio-nitin31.netlify.app/)

---

## Features

- **Modern UI/UX Design**: Clean, responsive, and visually appealing interface.
- **Showcase Projects**: Interactive project cards with links to GitHub repositories or live demos.
- **Skills Section**: Highlights technical skills with icons and animations.
- **Contact Form**: Users can get in touch via a functional contact form.
- **Responsive Design**: Optimized for desktop, tablet, and mobile screens.
- **Smooth Animations**: Implemented using GSAP for engaging interactions.

---

## Tech Stack

- **Frontend:** React, Vite
- **Styling:** Tailwind CSS
- **Animations:** GSAP
- **Deployment:** Netlify
